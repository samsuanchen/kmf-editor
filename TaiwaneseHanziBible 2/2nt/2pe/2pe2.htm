彼 得 後 書

章 1 2 3

2:1 若是佇百姓的中間曾有假的先知，按呢佇恁的中間也啲欲有假的先生，暗靜引進許個害人到沈淪的異端入來，就贖回 的主宰也呣認，家己取緊緊到的沈淪。
2:2 欲有多多人趁的邪淫，真理的路欲為著來受誹謗。
2:3 對貪心欲造作話來(lut)恁。對早以來的審判無延延(iân-chhiân)，的沈淪無歇睏。
2:4 因為上帝既然無惜犯罪的天使，是佇暗府，交佇黑暗的坑，收禁啲聽候審判。
2:5 閣無惜古早的世界，互大水到佇許個無敬虔的人的世界，獨獨保護宣傳義理的挪亞彼八人。
2:6 閣互所多瑪、蛾摩拉許個城，做火灰，擬定滅無，下啲做後世代無敬虔的人的鑑戒；
2:7 獨獨拯救義人羅得，就是常常為著誣理的人邪淫的所行來深深憂悶的。（
2:8 因為此個義人徛起佇中間，也看見也聽見日日因為不法的所做，來苦楚伊義的心。）
2:9 主知通拯救敬虔的人脫離誘惑，也知通收禁不義的人到審判的日，受刑罰。
2:10 閣卡要緊是許個趁肉體行佇污穢的私慾，也藐視辦政事者。大膽自恃，就褻瀆 ( siat-tok ) 尊貴的，也無驚。
2:11 雖然天使氣力權能卡大，猶久未曾佇主的面前用謗瀆的話責罪。
2:12 獨獨諸個人親像無靈性的活物，生成著互人掠來敗壞的，褻瀆所呣知的，欲佇的敗壞來受敗壞。
2:13 抵著艱苦，做不義的工錢。掠日時的宴樂做歡喜，是垃墋，是瑕疵 ( hâ-chhû )，及恁同筵席，宴樂佇的和愛席。
2:14 的目睭攏是淫色，犯罪無息，迷惑心無堅固的人，的心慣勢貪財，合該受咒詛。
2:15 離開正路來失迷，趁比珥的子巴蘭的路。伊是憶 ( it ) 著不義的工錢，
2:16 若是為著家己的犯法來抵著責備；(bõe)講話的驢講人的話來阻止先知的濫糝。
2:17 無水的泉，狂風所掃的雲霧，有黑黑暗暗留啲聽候。
2:18 因為講空虛誇口的話，佇肉體的私慾用邪淫來誘惑許個抵抵脫離行差謬 ( chha-biü) 的人的手的。
2:19 應允自由，也家己做敗壞的奴才，因為人互甚麼卡贏就做彼項的奴才。
2:20 因為已經對深知救主──耶穌基督，來得著脫離世間許個污穢，若閣再互許個污穢所纏所壓制，就彼號人路尾的事比代先卡傷重。
2:21 因為既然知義的路，也竟然悖謬所傳互聖的命令，呣值著攏呣知閣卡好。
2:22 彼號人的景況是合佇真的俗語所講：狗倒轉食伊所吐的；豬已經洗閣淖佇土糜的(thô͘-bê )中央。
